# Home Lab: IT Portfolio and Income Pipeline

Building a real-world IT environment from scratch to develop job-ready skills, document proof, and convert that into career opportunities and income.

This project is not just about learning. It is about turning skills into proof, and proof into opportunities.

## Objective

Build a portfolio-backed IT skillset that leads directly to:

- Entry-level IT role (short-term)
- Freelance / side income (mid-term)
- Automation + systems engineering (long-term)

## Core Philosophy

- Learn by building, not consuming
- Everything becomes documented proof
- Focus on high-ROI skills only
- Convert projects into portfolio proof, then into income

## Workflow

Build -> Break -> Fix -> Document -> Publish

## Roadmap (Execution Order)

### Phase 1: Foundation

- Install Proxmox
- Configure networking
- Enable remote access

### Phase 2: Core Systems

- Deploy VMs
- Set up Docker
- Run initial services

### Phase 3: Real Scenarios

- User and system management
- Network troubleshooting
- Failure and recovery drills

### Phase 4: Optimization

- Monitoring and logging
- Automation scripts

### Phase 5: Portfolio to Income

- Publish polished documentation
- Apply for IT roles
- Offer freelance services

## Skills Obtained

- Infrastructure and virtualization
- Networking fundamentals
- System administration
- Containerization
- Troubleshooting and problem solving
- Monitoring and system awareness
- Automation
- Documentation and communication
